## `01` вҖ” Overview

Agency website for **Keystone Software Solutions** вҖ” a product and services studio based in Dhaka, Bangladesh. Built to reflect the precision and intentionality we bring to every client engagement.

Dark, kinetic, and built to ship. The site showcases our work, capabilities, and process вҖ” with a design language inspired by the Active Theory aesthetic: absolute black backgrounds, high-contrast typography, and motion that means something.

## `07` вҖ” Getting Started

**Clone and run locally:**

```bash
git clone https://github.com/munim430-ai/Keystone-Software-.git
cd Keystone-Software-
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

<br/>

**Build for production:**

```bash
npm run build
npm run preview
```

<br/>

## `08` вҖ” Deploy

The project is configured for **zero-config Vercel deployment.**

```bash
# Push to main вҶ’ Vercel auto-deploys
git add .
git commit -m "your message"
git push origin main
```

`vercel.json` handles SPA routing so all `/work`, `/services`, `/about`, `/contact` routes resolve correctly.
