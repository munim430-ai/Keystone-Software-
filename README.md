# Keystone Software Solutions

**Efficiency, Engineered.**

Agency website for Keystone Software Solutions — built with React 19, Vite, TypeScript, Tailwind CSS v4, GSAP, Framer Motion, and Lenis.

Live: [keystone-software.vercel.app](https://keystone-software.vercel.app)

---

## Stack

| Layer | Tool |
|-------|------|
| Framework | React 19 + Vite 6 (TypeScript) |
| Styling | Tailwind CSS v4 |
| Animation | GSAP + ScrollTrigger, Framer Motion |
| Smooth Scroll | Lenis |
| Font | Geist Variable |
| Routing | React Router v6 |
| Deploy | Vercel |

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy (Vercel)

1. Push to GitHub
2. Import repo in Vercel
3. Framework: **Vite**
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Deploy

The `vercel.json` handles SPA routing rewrites automatically.

## Pages

- `/` — Home (all sections)
